iTunesふりかけ

漢字タイトルのせいで曲の並び順がぐちゃぐちゃな方へ。
自動で読みがなを振ることで、曲の並び順をマシにするソフトです。

ダウンロード (itunes_furikake_v2_3_0.zip)
2020.07.16 release
ツイート
動作を確認した環境：Windows10
実行に必要なプログラム：
・iTunes → 探しに行く
・Microsoft .NET Framework 4.6 以上の Runtime (Windows10 標準装備のはず) → 探しに行く
・Microsoft IME (標準装備のはず)
